# TabBar 图标说明

## 需要准备的图标文件

请在 `static/tabbar/` 目录下准备以下图标文件：

### 1. 首页图标
- `home.png` - 未选中状态（灰色）
- `home-active.png` - 选中状态（蓝色）

### 2. 账单图标
- `bill.png` - 未选中状态（灰色）
- `bill-active.png` - 选中状态（蓝色）

### 3. 记录图标
- `record.png` - 未选中状态（灰色）
- `record-active.png` - 选中状态（蓝色）

### 4. 图表图标
- `chart.png` - 未选中状态（灰色）
- `chart-active.png` - 选中状态（蓝色）

### 5. 管理图标
- `manage.png` - 未选中状态（灰色）
- `manage-active.png` - 选中状态（蓝色）

## 图标规格建议

- 尺寸：81px × 81px（推荐）
- 格式：PNG（支持透明背景）
- 未选中颜色：#7A7E83（灰色）
- 选中颜色：#2979ff（蓝色）

## 图标设计建议

1. **首页（home）**: 房子图标
2. **账单（bill）**: 账单/文档图标
3. **记录（record）**: 编辑/铅笔图标
4. **图表（chart）**: 柱状图/折线图图标
5. **管理（manage）**: 设置/齿轮图标

## 临时方案

在没有准备好图标之前，可以：
1. 使用 iconfont 网站下载免费图标
2. 使用设计工具（如 Figma、Sketch）制作简单图标
3. 暂时使用文字模式（修改 pages.json 中的 tabBar 配置，去掉 iconPath）

## 不使用图标的配置方式

如果暂时不需要图标，可以在 `pages.json` 的 tabBar 配置中移除 `iconPath` 和 `selectedIconPath` 字段，只保留 `text` 字段即可显示纯文字导航。
